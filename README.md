# Simple Interest Calculator

## 📌 Description
This project calculates simple interest using a basic mathematical formula.

## 🧮 Formula
Simple Interest = (Principal × Rate × Time) / 100

## 💻 Example
If:
- Principal = 1000  
- Rate = 5%  
- Time = 2 years  

Then:
Simple Interest = (1000 × 5 × 2) / 100 = 100

## ▶️ Usage
Enter:
- Principal amount  
- Rate of interest  
- Time period  

The program will return the simple interest.
